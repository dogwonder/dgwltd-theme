Ola DGW.ltd

An open-source hybrid theme for WordPress. This leans heavily on block based theming, although is not a Full Site Editing (FSE) theme, rather a combination of custom blocks, theme.json and PHP templating, read more.

This site uses DeLorean ipsum for placeholder text, and some Back to the Future based fun.

There is also a custom plugin for this site that allows for a bunch of custom blocks. The blocks page uses many of these.

  • DGW.ltd Accordion – based on GOV.UK’s accordion pattern
  • DGW.ltd Banner – text and background image similar to hero but less showy
  • DGW.ltd Embed – lite embed custom element for Youtube and Vimeo
  • DGW.ltd Cards – grid of cards linking to other pages, title, excerpt and featured image
  • DGW.ltd Image – custom image with aspect ratio variables
  • DGW.ltd Hero – hero with big image / video as background
  • DGW.ltd Promo card – offset image and content block
